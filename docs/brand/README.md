## Quick Reference

### Primary Colors
- **Black**: `#000000` - Text, borders
- **White**: `#F5F5F5` - Backgrounds
- **Red**: `#C41E3A` - CTAs, accents

### Primary Fonts
- **Display**: Bebas Neue (headlines, buttons)
- **Body**: System fonts (paragraphs)
- **Data**: JetBrains Mono (numbers)

### Brand Voice
- Direct and confident
- No fluff, no filler
- Respect user's time and intelligence
